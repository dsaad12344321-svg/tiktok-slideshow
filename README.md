# TikTok Content Rewards Slideshow Generator

A premium React + Vite web application that generates high-quality Arabic TikTok slideshows for the **Content Rewards (Slideshows)** campaign. Generate 6-slide photo slideshows with luxury imagery, Arabic text, glassmorphism design cards, and a canvas-based 1080×1920 PNG export engine.

## Features

- ⚡ **One-click generation** — Generates completely new Arabic text, images and design each time
- 📥 **HD Download** — Exports all 6 slides as 1080×1920 PNG files (TikTok-ready)
- 🎨 **Full customization** — Font, overlay opacity, card style, hashtag, keyword
- 🇸🇦 **Arabic RTL support** — Proper right-to-left text rendering on canvas
- 🏎️ **Luxury image library** — Cars, yachts, watches, pools, skylines, resorts
- ✅ **Campaign-compliant** — Enforces all Content Rewards rules

## Slideshow Structure

| Slide | Purpose |
|-------|---------|
| 1 | Scroll-stopping hook with luxury image |
| 2–3 | Explain the opportunity |
| 4 | **Content Rewards** (CRITICAL — required for approval) |
| 5 | Why Clipping — benefits list |
| 6 | CTA — Comment keyword to trigger DM automation |

## Getting Started

```bash
npm install
npm run dev
```

## Build for Production

```bash
npm run build
```

## Deploy to Vercel

1. Push this repo to GitHub
2. Connect repo to Vercel at [vercel.com](https://vercel.com)
3. Vercel auto-detects Vite — deploy with zero configuration

## Campaign Rules

- Use `#contentrewardspartner` on every post
- **Never skip Slide 4** — missing it means rejection
- Add your referral link to your bio
- Set up ManyChat (TikTok/IG) or CommentShark (YouTube) automation
- Submit every video — no submission = no payout
